# REQUIREMENTS

* Multiple blog posts

* CRUD blog post

## Features


### URL's 

* GET /post -> show and index of all blog post

* GET /posts/:id -> show blog post by id

* GET /post/new -> a form for new blog posts

* POST /post -> submit new blog post

* GET /posts/:id/edit -> edit blog post
